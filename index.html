Question Tosser

An open source project I just started this semester. I came up the idea because there are a lot of students who are too shy to ask questions in class and later they forget about the questions. I believe asking the questions actively is the best way to learn. That's why there should be an application should help quiet students to ask questions more actively and openly.

Question Tosser is built with ASP.NET MVC and SignalR technologies with WebSockets under the cover. Other technologies used include: C#, Javascript, AJAX, JQuery, MySQL, etc.

Github link: https://github.com/mikeogod/QuestionTosser
